Iniciando uma aplicação django dentro do projeto

						
						$ cd .\cafecompao\
						$ \cafecompao> python manage.py startapp padarias

Criando primeira view

						
							# file padarias/views.py

							from django.shortcuts import render
							from django.http import HttpResponse

							def index(request):
									return HttpResponse("Bem vindo a rede de padarias Cafe com Pao")

Mapeando URLs

Criar arquivo urls.py no app 'padarias'

						
							# file padarias/urls.py
							from django.urls import path

							from . import views

							urlpatterns = [
									path('', views.index, name='index'),
							]

Redirecionando URLs

Alterar o arquivo global urls.py em `cafecompao/urls.py`

						
							# file cafecompao/urls.py
							from django.contrib import admin
							from django.urls import path, include

							urlpatterns = [
									path('admin/', admin.site.urls),
									path('', include('padarias.urls')),
							]
